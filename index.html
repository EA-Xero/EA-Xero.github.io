Tecnologias Aprendidas

HTML/CSS/JS

Lenguajes base para la programación y diseño web.

Bootstrap

Framework de CSS para el diseño web, ideal para agilizar el desarrollo y mejorar la consistencia visual en proyectos.

React.js

Librería de JavaScript para el desarrollo web, destacada por optimizar el tiempo de desarrollo y facilitar la gestión eficiente de la información en aplicaciones interactivas y dinámicas.

Node.js y Express.js

Plataforma y framework de JavaScript respectivamente, utilizados para el desarrollo de aplicaciones del lado del servidor. Node.js proporciona el entorno de ejecución, mientras que Express.js simplifica la creación de aplicaciones web robustas.

PostgreSQL

Sistema de gestión de bases de datos relacional, conocido por su robustez y capacidades avanzadas. PostgreSQL ofrece una sólida base para almacenar y gestionar datos, siendo una opción preferida en aplicaciones que requieren integridad y rendimiento.

Figma

Herramienta de diseño colaborativo basada en la nube. Figma es reconocida por su capacidad para simplificar el proceso de diseño y prototipado, permitiendo a los equipos trabajar de manera eficiente en proyectos creativos.
