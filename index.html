Tecnologias

  • PostgreSQL

    Importante motor de bases de datos el cual utiliza como lenguaje el famoso SQL, permite realizar bases de datos relacionales y es mundialmente conocido y estandar en el mercado

  • HTML , CSS , JS

    Stack basico de desarrollo web completamente necesario para paginas basicas e intermedias. perfecto para maquetacion de paginas web, sobre todo landing pages y presentaciones

  • React JS

    Libreria que utiliza JavaScript y/o Typescript, ideal para paginas que ameritan mas trabajo como tiendas o proovedores de servicios

  • Node JS

    Es un motor nativo de JS el cual permite realizar trabajos mas complejos como operaciones CRUD y servidores sin cambiar de lenguaje como lo es JavaScript

  • BootStrap \ TailWind

    Son Herramientas que permiten facilitar el diseño de componentes y paginas web mediante decoracion y definicion de comportamientos mediante clases
